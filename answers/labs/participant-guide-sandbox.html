Hands-On Lab

Team Based Rapid Prototyping and building AI Agents
with Cortex Code CLI

Use a custom skill to build a sandbox prototyping environment for natural language querying of your data

Prerequisites

  • Continuation of the previous lab.
  • Cortex Code CLI installed and configured with .toml configuration file for user with ACCOUNTADMIN role
  • FROSTBYTE_DB.MARTS database.schema present with two tables populated
  • sandbox-frostbyte folder/skill copied to ~/.snowflake/cortex/skills
Tip: Ask CoCo if it can find the skill to make sure it's present and aware of it.
1

Provision your prototyping Sandbox Environment

10 min
Key Concept — Skills

A user in a particular business unit can follow a custom skill to clone data for a specific business unit in order to work on that data and perform tasks such as building an AI Agent.

Ask the following question of CoCo to invoke the skill...

Create a sandbox environment
Expected result: This should invoke our custom skill which will initially prompt you to select a team/business unit

For this lab, please select Retail / Store Operations

The skill confirm your selections and the names of the database objects.

Proceed with the DDL creation.

Expected result: Sandbox will be provisioned and you can see the objects in Snowsight

2

Generate Analytics Prototype (Semantic View and Cortex Analyst and Cortex Agent)

20 min
Key Concept — Skills Workflow

A skill can provide a workflow and based on user selections can step through a process to generate Snowflake objects. In this case we want to select the option for an 'Analytics Prototype'

Continue on with the skill workflow and Select/type option E

When prompted for the business questions you want to answer select the general purpose model

CoCo will continue on and generate the yaml and semantic view and test it. Once complete you can see the questions and answers. You can test the Cortex Analyst semantic model with an additional question such as:

Test the semantic view with this question: Which suppliers have the most products below reorder point?
Expected Result: A list of suppliers and a count of products below reorder.

You can also view the semantic view in Snowsight under AI & ML - Cortex Analyst - Semantic View - SANDBOX_FROSTBYTE_RETAIL_POC.MARTS

Ask CoCo to create a Cortex Agent based on this semantic view now we know it works.

Create a Cortex Agent based on this semantic view
Expected result: Successful creation of the agent.

You can go into Snowsight - AI & ML - Agents and find your agent and 'Preview in Snowflake Intelligence' and ask questions of the data

3

Build a Streamlit app to display the data

15 min
Key Concept — Bundled Skills

CoCo can be used to create a streamlit app/dashboard on your data in Snowflake

Let's try another prototyping track and select building a streamlit app

I'd like to choose another prototyping track

Select Option C: Prototype App

In this case we are instructing CoCo to parse the image file, read the general style of the example dashboard and use that as a basis for our new Streamlit dashboard.

You could build into your skill the corporate style you wish all Streamlit apps to follow.

CoCo will suggest some data to visualise - Select Choose Something else and type....

Display Inventory Health Dashboard data and use the ~/.snowflake/cortex/skills/sandbox-frostbyte/streamlit_template.png image as a guide for the style of the dashboard

CoCo will now create a compute pool and the external access integration (for any python libraries it needs). It will also create the file artifacts locally and then deploy them into Snowflake.

Expected result: Click on the link provided by CoCo and view the Streamlit app.

Optional: Continue on and have CoCo make changes to the Streamlit app...
